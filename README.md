# qr-generator

Basic qr-generator application

**Dependencies:** `qrcode`, `Tailwind` 

## Generator Module
```js
      QRCode.toDataURL('Crazy text!')
      .then(url => {
        console.log(url)
        })
      .catch(err => {
        console.error(err)
      })
```
